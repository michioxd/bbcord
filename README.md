# BBCord

Unofficial Discord client for QNX BlackBerry 10 devices. It is built using Qt and the Discord API.

## Note

This project still in early stages of development, and many features are not yet implemented. The app is currently in a very rough state, and may be unstable. Use at your own risk.

## Installation

- Because this app is unsigned, you will need to root your BB10 device first.
- Download the latest `BBCord-<version>.bar` file from the [releases page](https://github.com/michioxd/bbcord/releases/latest)
- Install the .bar file using [Sachesi](https://github.com/xsacha/sachesi) or DBBT.
- Launch the app and log in with your Discord token.

## How to get Discord token

[Google it](https://www.google.com/search?q=how+to+get+discord+token+github+gist)
